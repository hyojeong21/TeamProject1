> **도서 검색부터 상세 조회, 장바구니, 회원 기능까지 제공하는 온라인 서점 웹 서비스**

국내도서, 해외도서, eBook, 중고도서를 카테고리별로 조회하고, 도서 검색과 장바구니, 회원 기능을 제공하는 **Vanilla JavaScript 기반 웹 프로젝트**입니다.

# 🎯 서비스 목표

검색을 통해 원하는 도서를 빠르게 찾는 사용자와, 다양한 도서를 둘러보며 탐색하는 사용자의 이용 방식을 모두 고려하여 서비스를 설계했습니다.

검색창, 네비게이션 바, 장르별 사이드바, 베스트 도서 UI를 중심으로 사용자가 원하는 도서를 쉽고 편리하게 탐색할 수 있는 웹 서비스를 구현하는 것을 목표로 했습니다.

# 💡 주요 구현 내용

## 1️⃣ 카테고리별 도서 관리

국내도서, 해외도서, eBook, 중고도서를 각각 분리하여 관리하고,

사용자가 원하는 카테고리를 선택하면 해당 목록만 동적으로 출력되도록 구현했습니다.

---

## 2️⃣ JavaScript 기반 동적 화면 구성

Vanilla JavaScript를 활용하여

- 도서 목록 출력
- 상세 페이지 이동
- 검색 결과 출력
- 장바구니 기능

등을 구현했습니다.

---

## 3️⃣ 장바구니 기능 구현

사용자가 선택한 도서를 장바구니에 담고

- 수량 변경
- 금액 계산
- 상품 삭제

등의 기능을 구현하여 실제 쇼핑몰과 유사한 사용자 경험을 제공했습니다.

---

## 4️⃣ 회원 기능 구현

회원가입과 로그인 기능을 구현하고,

마이페이지에서 사용자 정보를 조회할 수 있도록 구성했습니다.

---

## 5️⃣ 재사용 가능한 공통 UI 구성

Header, Navigation 등 공통 영역을 분리하여

중복 코드를 줄이고 유지보수성을 높였습니다.

# 🚀 실행 방법

### 1. VS Code에서 프로젝트 열기

프로젝트를 Visual Studio Code에서 실행합니다.

### 2. Live Server 실행

1. `index.html` 파일을 엽니다.
2. **Live Server** 확장 프로그램이 설치되어 있지 않다면 설치합니다.
3. 우측 하단 또는 마우스 우클릭 메뉴의 **Go Live**를 클릭합니다.

브라우저가 자동으로 실행되며 프로젝트를 확인할 수 있습니다.

> **또는** `index.html` 파일을 브라우저에서 직접 실행하여 확인할 수도 있습니다.
